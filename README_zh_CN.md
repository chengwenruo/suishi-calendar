# 岁时日历（SuiShi Calendar）

岁时日历是一个更符合国人习惯的日历面板插件，支持公历/农历、法定节假日、二十四节气与日记快速创建/跳转。

## 功能

- 月历视图（含 ISO 周数）
- 显示农历；当日有节日/节气时优先显示节日/节气
- 内置 2026 年法定节假日与调休标记
- 点击日期创建或打开对应日记
- 按日记字数显示 1-3 个圆点（阈值可配置）
- 字体模式：跟随 SiYuan / 跟随侧边栏 / 固定大小
- 季度显示：数字季度或春夏秋冬
- 界面风格：卡片风格 / 极简风格

## 设置入口

- 思源：`设置 -> 集市 -> 已安装 -> 岁时日历 -> 设置`
- 顶栏按钮：`岁时日历设置`
- 命令面板：`打开岁时日历设置`
